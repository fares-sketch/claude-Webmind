# 08 - VARIANTS STRATEGY (A/B TESTING)

Le système est conçu pour générer des dizaines de publicités. Voici comment créer des variantes pour toucher différents segments de clientèle tout en réutilisant 70% de la vidéo de base.

## L'Architecture Modulaire
La publicité est composée de "Briques" (Séquences) interchangeables.

**Modèle de base :**
[Intro] + [Valeur 1] + [Valeur 2] + [Valeur 3] + [Outro]

### Variante 1 : L'Angle "Gain de Temps" (Pour les entrepreneurs/cadres)
- **Même** : Intro, Outro, Livraison.
- **Nouvelle Brique** : Insister sur l'interface fluide ("L'achat en 3 clics") et la gestion locative ("Une comptabilité sans friction pour votre société").
- **Visuel de Remplacement IA** : Générer une tablette posée sur une table en verre fumé, reflétant le building du bureau.

### Variante 2 : L'Angle "Dépannage / Zéro Tracas"
- **Même** : Intro, Outro, Offre sur mesure.
- **Nouvelle Brique** : Insister sur le volet entretien. "Un voyant s'allume ? On vient chercher le véhicule."
- **Visuel de Remplacement IA** : Clé remise sur un comptoir dans un atelier futuriste, avec en arrière-plan un mécanicien (silhouette floue) inspectant le pneu.

### Variante 3 : Renouvellement de Flotte / Offre Multivolant
- **Même** : Outro, Interface, Livraison.
- **Nouvelle Brique Intro** : Un lineup / garage avec non pas une seule Porsche, mais 3 véhicules garés en parallèle (ex: Une Porsche 911, Un Range Rover, Une Audi RS6).
- **Voix Off modifiée** : "À chaque moment de votre vie correspond un véhicule. Prestige Motors adapte votre garage selon vos envies de la semaine."

## Process d'itération CapCut
1. Dupliquez le projet Master dans CapCut.
2. Remplacez uniquement la voix off (Piste A1) et les clips vidéo concernés (Piste V1).
3. Adaptez le texte (Piste V3).
4. Exportez. -> Vous avez une nouvelle Ad très qualitative en 10 minutes.
