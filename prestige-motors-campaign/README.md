# Prestige Motors - Campagne Verticale Premium

## Vision Globale
Système de production pour la marque **Prestige Motors**, une conciergerie automobile haut de gamme.
L'objectif est d'établir un workflow de production vidéo réutilisable pour générer des publicités verticales (9:16) via IA (Sora, Runway, Kling), doublées via ElevenLabs et finalisées sur CapCut.

## Positionnement & Direction Artistique
- **Mots-clés** : Luxe discret, simplicité, service premium, fluidité, gestion complète, livraison sur mesure, transparence.
- **Esthétique** : Noir, blanc, gris, argent (silver).
- **Vibe** : Minimaliste, architectural, mouvements lents et maîtrisés.
- **Interdits** : Bling bling, trading vibe, montage frénétique, interfaces sci-fi, texte généré par l'IA dans la vidéo (ajout en post-prod uniquement), pas de motion agressive, pas d’énergie racing, pas de présence humaine visible.

## Arborescence du Projet
- `/00_master` : Fichiers de référence et vision globale.
- `/01_brand_direction` : Règles strictes de la DA de la marque pour la constance des assets.
- `/02_storyboards` : Storyboards séquence par séquence.
- `/03_prompts_sora` : Bibliothèque de prompts optimisés pour OpenAI Sora.
- `/04_prompts_runway` : Bibliothèque de prompts optimisés pour Runway Gen-2 / Gen-3.
- `/05_prompts_kling` : Bibliothèque de prompts pour Kling.
- `/06_voiceover` : Scripts et références pour ElevenLabs.
- `/07_capcut_editing` : Plans et directives de montage.
- `/08_variants` : Déclinaisons de l'annonce principale (livraison, assurance, etc.).
- `/09_assets_notes` : Notes annexes, assets graphiques et typos.

## Workflow de Production
1. Choisir le storyboard et la variante.
2. Générer les assets vidéo selon les prompts (Sora / Runway / Kling).
3. Générer la voix off via ElevenLabs.
4. Assembler selon la structure de montage CapCut.
5. Ajouter les textes en surimpression selon la typographie de la marque.
