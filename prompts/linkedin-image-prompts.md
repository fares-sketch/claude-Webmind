## Instructions d'utilisation

1. Copie le prompt voulu
2. Va sur **ChatGPT → DALL·E** ou **Midjourney** ou **Adobe Firefly**
3. Colle le prompt
4. Télécharge le résultat et enregistre dans `/claude_code/assets/linkedin/`

## Conseil
Pour garder la cohérence visuelle entre toutes les images :
> Commence toujours par le prompt **Avatar (n°10)** pour établir le style de base, puis génère les autres en référençant "même style que l'image précédente" dans Midjourney.
